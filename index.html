Built To Disagree

Most AI tools agree with everything. AIDEN's phantoms include confrontation directives.

CONFRONTATION PHANTOMS

CHALLENGE_BACK, REJECT, PUSH_BACK, DEFEND phantoms activate when the user needs honest pushback, not agreement.

ESCALATION DETECTION

When a user asks for "bolder", bold phantoms get a 1.5× boost. The system responds to creative hunger.

CREATIVE RESET

When an idea is killed, previously-dominant phantoms get suppressed (0.3×), forcing genuinely new thinking.

KILL DETECTION

Prevents recycling the same idea with synonyms. If you killed it, it stays dead. New territory only.
